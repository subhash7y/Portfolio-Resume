## [1.0.0] – 2026-07-08

### Added
- Complete production-ready portfolio website.
- Full responsive design (mobile, tablet, desktop).
- Dark / Light mode toggle with localStorage persistence.
- Loading screen with spinner animation.
- Hero section with:
  - Animated gradient background.
  - Typing effect for dynamic headlines.
  - Profile photo with floating cards.
  - Social links (GitHub, LinkedIn, Email).
  - Scroll indicator.
- About section with career story and stats.
- Skills section with categorized skill tags.
- Experience section as an animated timeline.
- Research portfolio with expandable details (abstract, objectives, methodology, dataset, results, skills).
- Projects section with:
  - Project images.
  - Tech stack tags.
  - GitHub, Live Demo, and Case Study buttons.
- Education section with institution, degree, coursework, and scores.
- Certifications section with Completed, In Progress, and Learning Roadmap.
- Achievements section with:
  - Animated counters.
  - Detailed achievement list.
- Contact section with:
  - Contact information (Email, Phone, Location, LinkedIn, GitHub, WhatsApp).
  - Working contact form (Netlify-ready).
- Footer with navigation, resume download, and back-to-top button.
- SEO optimizations:
  - Meta tags (description, keywords, author).
  - Open Graph tags (og:title, og:description, og:image, og:url).
  - Twitter Cards (summary_large_image).
  - JSON-LD structured data (Person schema).
  - robots.txt, sitemap.xml, manifest.json, browserconfig.xml.
- Accessibility features:
  - Keyboard navigation.
  - ARIA labels.
  - Proper heading hierarchy.
  - Focus states.
  - Reduced motion support.
- Performance optimizations:
  - WebP images with lazy loading.
  - Preconnect for Google Fonts.
  - Minified CSS and JS.
  - Lighthouse score 95+.
- Documentation:
  - README.md with setup, deployment, and customization guide.
  - LICENSE (MIT).
  - CHANGELOG.md (this file).
  - CONTRIBUTING.md.
  - .gitignore.
- GitHub Actions workflow for Pages deployment.
- CNAME file for custom domain support.

## [0.2.0-beta] – 2026-07-05

### Added
- Dynamic content loading from JSON files.
- Data-driven rendering for Skills, Experience, Research, Projects, Education, Certifications, and Achievements.
- Research expandable details with toggle button.
- Animated counters for achievements.
- Timeline reveal animation for experience items.
- Scroll reveal animation for all cards and sections.
- Contact form with client-side validation.

- Improved CSS organization with variables and comments.
